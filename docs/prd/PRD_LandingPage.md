# Landing Page PRD - 本周吃什么 (EatWhat AU)

## 文档信息
- **产品名称**：本周吃什么 / What to Eat This Week
- **目标市场**：澳大利亚
- **文档版本**：v1.0
- **创建日期**：2026-01-13
- **文档类型**：Landing Page 产品需求文档

---

## 1. Landing Page 目标

### 1.1 核心目标
- **主要转化目标**：引导用户注册/开始使用产品
- **次要目标**：传达产品核心价值，建立品牌认知

### 1.2 目标用户（澳洲市场）
1. **在澳华人家庭**（主要）- 希望用本地食材做中餐
2. **留学生** - 预算有限，需要省钱又健康
3. **年轻家庭** - 注重营养均衡和便利性
4. **健康饮食爱好者** - 关注食材质量和营养搭配

### 1.3 页面关键指标（KPI）
- 注册转化率：>15%
- 页面停留时间：>45秒
- CTA点击率：>20%
- 跳出率：<60%

### Section 8: FAQ（常见问题）

#### 标题
"常见问题"

#### 问题列表（可折叠）

**Q1: 这个服务是免费的吗？**
A: 基础功能完全免费。高级功能（如营养分析、无限食谱等）可升级到会员版。

**Q2: 支持哪些超市？**
A: 目前支持 Woolworths、Coles、ALDI、IGA 和 Costco。我们会持续添加更多超市。

**Q3: 食谱都是中餐吗？**
A: 以中餐为主，同时也有西餐、日韩料理、东南亚菜等多种选择。

**Q4: 价格信息多久更新一次？**
A: 每天自动更新超市价格和优惠信息。

**Q5: 可以设置饮食限制吗？**
A: 可以。支持素食、无麸质、低糖、过敏原筛选等多种饮食限制。

## 3. 文案要点

### 3.1 核心卖点（Value Propositions）
1. **省钱** - "每周节省 $50+，聪明采购不浪费"
2. **省时** - "3分钟规划一周，告别做饭焦虑"
3. **健康** - "营养师认证，均衡搭配更健康"
4. **本地化** - "用澳洲食材，做出家的味道"

### 3.2 文案风格
- 友好、亲切、生活化
- 中英文结合（主要中文，关键处英文）
- 突出澳洲本地化特点
- 强调实用价值和情感连接

## 4. 视觉设计要求

### 4.1 色彩方案
| 颜色用途 | 色值参考 | 说明 |
|----------|----------|------|
| 主色调 | #4CAF50（绿色） | 健康、新鲜感 |
| 辅助色 | #FF9800（橙色） | 食欲、活力 |
| 背景色 | #FFFFFF / #F5F5F5 | 干净、简洁 |
| 文字色 | #333333 / #666666 | 易读性 |
| CTA按钮 | #FF6B35 或 #4CAF50 | 高对比度 |

### 4.2 图片要求
- **Hero图片**：高质量家常菜，有烟火气
- **功能配图**：简洁的图标或插画
- **超市Logo**：官方高清Logo
- **食谱图片**：真实食物照片，不要过度PS

### 4.3 字体
- **中文字体**：思源黑体 / PingFang SC / 微软雅黑
- **英文字体**：Roboto / Open Sans / Inter
- **标题**：Bold / 700
- **正文**：Regular / 400

## 5. 响应式设计

### 5.1 断点设置
- **Mobile**：< 768px（优先设计）
- **Tablet**：768px - 1024px
- **Desktop**：> 1024px

### 5.2 移动端优化
- 单列布局
- 大号可点击按钮（最小 44x44px）
- 简化导航
- 图片懒加载
- 触摸友好的交互

---

## 6. 技术实现建议

### 6.1 技术栈
- **HTML5** - 语义化标签
- **CSS3** - Flexbox/Grid 布局
- **原生 JavaScript** 或 **轻量框架**（如 Alpine.js）
- **响应式框架**（可选）：Tailwind CSS / Bootstrap

### 6.2 性能要求
- 首屏加载时间：< 2秒
- 总页面大小：< 3MB
- 图片优化：WebP 格式，懒加载
- 移动端优先

### 6.3 SEO优化
- 语义化 HTML 标签
- Meta 标签优化（title, description, keywords）
- Open Graph 标签（社交分享）
- 结构化数据（Schema.org）
- 图片 alt 属性

## 7. 交互设计

### 7.1 动画效果
- **滚动动画**：section 进入视口时淡入
- **按钮悬停**：轻微放大 + 颜色变化
- **卡片悬停**：阴影加深
- **平滑滚动**：锚点跳转平滑过渡

### 7.2 CTA 按钮交互
- 明确的视觉层次（主次分明）
- 悬停状态反馈
- 点击后状态变化
- Loading 状态提示

## 8. 转化优化策略

### 8.1 CTA 按钮布局
- Hero Section：1个主CTA + 1个次CTA
- 中间内容区：每个 section 后可选择性添加 CTA
- 最终 CTA Section：大号、居中、高对比度

### 8.2 信任元素
- 用户评价（真实头像 + 姓名 + 地点）
- 使用数据（用户数、食谱数、节省金额）
- 支持的超市 Logo（官方授权）
- "免费"、"无需信用卡"等承诺

### 8.3 降低决策门槛
- "完全免费"标签
- "无需信用卡"承诺
- "3分钟完成设置"时间承诺
- "随时取消"退出保证
